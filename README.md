## 🏆 Featured Project

### 🏦 HS Bank - Digital Banking & Loan Management API  
https://github.com/Henr1queSantos/HsBank  

A backend system built with **.NET 8**, designed to simulate a real banking environment using modern architectural principles.

This project focuses on building a realistic and scalable backend with clean separation of concerns and support for event-driven workflows.

#### ⚙️ Tech Stack
- C# 12 / .NET 8 ASP.NET Core  
- PostgreSQL + Entity Framework Core (Code-First)  
- RabbitMQ + MassTransit (Messaging)  
- JWT Authentication + BCrypt  
- xUnit, Moq, FluentAssertions  
- Docker + Docker Compose  

#### 🧠 Architecture Highlights
- Clean Architecture  
- CQRS with MediatR  
- Domain-Driven Design (practical approach)  
- Event-driven architecture (RabbitMQ)  
- Modular and scalable design
